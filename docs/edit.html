Einmalig — GitHub-Token erstellen (bleibt nur lokal im Browser, localStorage; „not accessible" heißt fast immer: falscher/zu schwacher Token):
  • Empfohlen, einfach: klassischen Token mit Scope repo anlegen → Generate token → ghp_… kopieren. (Ein Häkchen, funktioniert sicher.)
  • Alternativ, fein-granular: neuen Token → Repository access = Only select repositories → pleuston/limesblatt-edition auswählen → Permissions → Repository permissions → Contents = Read and write.
